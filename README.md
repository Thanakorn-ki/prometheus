# Prepare after run dashboard
 - install docker

# How to run dashboard grakana
> run
```sh
docker-compose up
```

> stop
```sh
docker-compose down
```
> The dashboard
- <localhost:3000> Grafana dashboard user:admin, password:admin
- <localhost:9000> Prometheus dashboard
